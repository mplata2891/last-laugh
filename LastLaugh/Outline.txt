Class Hierarchy

*GamePiece (Highest Level Superclass)

	@ Selector [is a GamePiece]
	
	@ Navigator [is a GamePiece]
		- has Menu(s)
		
		@ Menu [is a GamePiece]

	@ PositionTracker [is a GamePiece]
		- has Room(s)
		
		@ Room
			- has Door
			- has Caretaker
			- has PuzzlePiece(s)
			
			@ Door
				- has Note
				
			@ Item [is a GamePiece]	
				# Caretaker [is an Item]
					- has Key
					# Key [is an Item]
				# PuzzlePiece [is an Item
		
	@ Person [is a GamePiece]
		# Player [is a Person]
			- has Command(s)
		
			@ Command [is a GamePiece]
				- has move
				- has sense
				- has examine
				- has interact
				- has action
				- has inventory
				- has notebook
				@ Move 
				@ Sense 
				@ Examine 
				@ Interact 
				@ Action 
				@ Inventory 
				@ Notebook
